Nhập Hàng Loạt (File CSV)

Hỗ trợ nhập nhanh dữ liệu. Vui lòng chuẩn bị file Excel và lưu dưới dạng .CSV (UTF-8).

Cấu trúc cột (11 cột):
TenSP, Gia, SoLuong, DonVi, RAM, CPU, ROM, ManHinh, VGA, Khac, TenAnh
Ví dụ:
Laptop Dell,20000000,10,Cai,8GB,Core i5,256GB,14 inch,Onboard,Vo nhua,dell.jpg
* Lưu ý: Dữ liệu được đọc trực tiếp từ file CSV và cập nhật tự động vào database.
